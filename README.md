# Ultimate TaskMaster

Простое, но мощное приложение для управления задачами с:
- Темной/светлой темой
- Локальным сохранением
- Фильтрацией задач
- Анимациями

## Установка
1. Клонируйте репозиторий
2. Откройте `index.html` в браузере

## Особенности
- Адаптивный дизайн
- Быстрые анимации
- Поддержка touch-устройств
